\chapter{Heurísticas}
\label{ch:heuristicas}

As heurísticas são algoritmos úteis para encontrar soluções boas em problemas de larga escala, cujos algoritmos tradicionais levariam muito tempo.
Neste trabalho, focamos nossa implementação em duas heurísticas: Nearest Neighbour e Nearest Insertion.

Por outro lado, as Buscas Locais são algoritmos interessantes para melhorar uma solução já existente.
Neste trabalho, combinamos a busca local Swap com a heurística Nearest Neighbour para alcançar uma melhor solução.

\subsection*{Nearest Neighbour (Vizinho Mais Próximo)}
